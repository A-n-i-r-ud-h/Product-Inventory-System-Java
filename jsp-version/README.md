# JSP-Based Product Inventory System

## Description
This is the web-based version of the Product Inventory Management System developed using JSP and Servlets.
It enables users to manage products, update stock levels, and perform CRUD operations through a dynamic and user-friendly interface.
The system follows MVC architecture and uses JDBC for database connectivity.

## Technologies Used
- Java
- JSP
- Servlets
- JDBC
- MySQL

## Features
- Add, update, delete products
- Manage inventory stock
- View product details

## Concepts Used
- MVC Architecture
- DAO Design Pattern
- CRUD Operations

## How to Run

1. Clone the repository
2. Import project into Eclipse / IntelliJ
3. Configure Apache Tomcat server
4. Set up MySQL database and update DB credentials
5. Run the project on server
